# Gradient Descent & Linear Regression Project
## Overview

### This project demonstrates gradient descent from scratch and applies it to simple linear regression, both in Python and R. It is designed to help understand:

- How gradient descent works to find local minima of functions
- How linear regression can be implemented from scratch using the gradient descent algorithm
